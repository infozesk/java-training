Portée des variables

Les paramètres natifs sont copiés

Les objets sont passés en référence

E02_6

natifs: passage par copie


                        public static void main(String[] args) {
                            // passage par copie pour les types natifs: int / bool / String
                            int unEntier = 0;
                            // On va tenter de modifier a dans la méthode
                            passageParametre(unEntier);
                            // Au retour unEntier vaut toujours 0
                        }
                        public static void passageParametre(int unEntier) {
                            unEntier = unEntier + 1;
                        }
E02_6

tableaux et objets: passage par reférence


                        public static void main(String[] args) {
                            // passage par référence pour les objets
                            Point unPoint = new Point(0, 0);
                            passageParametre(unPoint);
                            //unPoint a été modifié !
                            // il vaut maintenant (100, 0)
                        }
                        public static void passageParametre(Point unPoint) {
                            unPoint.x = 100;
                        }
